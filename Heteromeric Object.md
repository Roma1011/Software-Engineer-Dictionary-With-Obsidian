### განსაზღვრება:

> **Heteromeric Object** — არის ისეთი ობიექტი,  
> რომელიც შედგება **განსხვავებული ტიპის ნაწილებისგან**,  
> რომელთაც განსხვავებული როლები ან ფუნქციები აქვთ მთლიანობაში.

ანუ ეს არის **ჰეტეროგენური კომპოზიტი**.

---

###  **მაგალითები:**

- ძაღლი → შედგება თავისგან, სხეულისგან, კუდისგან და ფეხებისგან.
    
- მანქანა → შეიცავს ძრავს, ბორბლებს, კორპუსს, საჭეს და ა.შ.
    
- ელფოსტის წერილი → შეიცავს ჰედერს და ტექსტურ აბზაცებს.
    
- `Car` კლასში:
    
    `class Car {     public Engine Engine { get; set; }     public Wheel[] Wheels { get; set; }     public Body Body { get; set; } }`
    
    `Car` არის **heteromeric object**,  
    რადგან მისი კომპონენტები სხვადასხვა ტიპისაა (`Engine`, `Wheel`, `Body`).
