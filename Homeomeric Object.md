###  განსაზღვრება:

> **Homeomeric Object** — არის ისეთი ობიექტი,  
> რომლის ყველა შემადგენელი ნაწილი **ერთნაირი ტიპისა და ბუნებისაა**.

ანუ მისი კომპონენტები **ჰომოგენურია (ერთგვაროვანია)**.  
თუ ობიექტს გავყობთ ნაწილებად, ყოველი ნაწილი იგივე ტიპის ობიექტია, რაც მთლიანია.

---

### **მაგალითები:**

- ქვიშის გროვა → ყველა ნაწილი ქვიშის მარცვალია.
    
- წყლის ჭიქა → ყველგან წყალია, ერთნაირი ნივთიერება.
    
- `List<int>` → ყველა ელემენტი ერთნაირი ტიპისაა (`int`).
    
- 3D Mesh-ის ვერტექსების სია → ყველა ელემენტი `Vertex` ტიპისაა.
    

---

###  პროგრამულად:

`class Fleet {     public List<Car> Cars { get; set; } = new(); }`

`Fleet` არის **homeomeric object**,  
რადგან ის შედგება მხოლოდ ერთი ტიპის კომპონენტებისგან — `Car` ობიექტებისგან.
