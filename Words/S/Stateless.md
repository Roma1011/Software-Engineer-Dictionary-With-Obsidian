**Stateless** ობიექტს ან სისტემას **არ აქვს შიდა მდგომარეობა**,  
ან არსებობს, მაგრამ ყოველი ოპერაცია **არ არის დამოკიდებული წინა ოპერაციებზე**.

ანუ:

> stateless ობიექტს **არ ახსოვს წარსული**.

---

###  მახასიათებლები:

- მეთოდები მხოლოდ არგუმენტებზეა დამოკიდებული.
    
- ერთი მოთხოვნის პასუხი არ არის დამოკიდებული წინა მოთხოვნებზე.
    
- მარტივი ტესტირება, მაღალი scalability (მიკროსერვისებში ძალიან პოპულარული).
    

---

###  მაგალითები:

- **Pure Functions** → `int Add(int a, int b)`
    
- **JWT-ზე დაფუძნებული ავტორიზაცია** → სერვერი Entity არ ინახავს.
    
- **HTTP (REST) APIs** → თითოეული რექვესთი დამოუკიდებელია.
    
- **Static Helper Classes** → utility ფუქნციები.


```csharp
static class MathUtil
{
    public static int Add(int a, int b)
        => a + b;  // does not depend on past operations
}
```
