**Port Address Translation (PAT)** არის **NAT-ის (Network Address Translation)** ტიპი, რომელიც საშუალებას აძლევს **ბევრ შიდა (private) მოწყობილობას** ერთდროულად გამოვიდეს ინტერნეტში **ერთი საჯარო IP მისამართის** გამოყენებით, **პორტების** დახმარებით.

ხშირად მოიხსენიება როგორც **NAT Overload**.


### როგორ მუშაობს

- შიდა ქსელის თითოეულ კავშირს ენიჭება **უნიკალური source port**.
    
- Router/NAT მოწყობილობა ინახავს ცხრილს, სადაც აკავშირებს:
    
    - **Private IP : Private Port** ↔ **Public IP : Public Port**
        
- დაბრუნებული ტრაფიკი პორტის მიხედვით სწორ შიდა მოწყობილობას უბრუნდება.

შიდა ქსელი:

- PC1 → `192.168.1.10:50001`
    
- PC2 → `192.168.1.11:50002`
    

საჯარო IP:

- `203.0.113.5`
    

NAT/PAT ცხრილი:

`192.168.1.10:50001 → 203.0.113.5:40001 192.168.1.11:50002 → 203.0.113.5:40002`
