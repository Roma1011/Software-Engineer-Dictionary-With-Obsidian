ნიშნავს _შემადგენელ ობიექტს_ ანუ ისეთ ობიექტს, რომელიც შედის სხვა უფრო დიდი ობიექტის შემადგენლობაში.

დავყოთ ნაწილებად:

- **Constituent** ნიშნავს „შემადგენელი“, „ნაწილი“,

ამრიგად, **constituent object** არის ობიექტი, რომელიც სხვა ობიექტის ნაწილია — მაგალითად, აგრეგატში.

---
 **მაგალითი (ობიექტზე ორიენტირებულ მოდელირებაში):**

თუ გვაქვს **Car** როგორც aggregate object  
და მასშია **Engine**, **Wheel**, **Door**,

მაშინ:

- `Car` არის **aggregate object (მთელი)**
    
- `Engine`, `Wheel`, `Door` არიან **constituent objects (ნაწილები)**
    

---

**ტერმინოლოგიურად**:

- aggregate object = მთლიანი ობიექტი (the “whole”)
    
- constituent object = შემადგენელი/კონსტიტუენტური ობიექტი (the “part”)
    

ასე რომ, როცა ამბობ:

> “An aggregate object is composed of constituent objects”

ეს ნიშნავს:

> „აგრეგატური ობიექტი შედგება კონსტიტუენტური (შემადგენელი) ობიექტებისგან.“ 

გინდა აგიხსნა განსხვავება constituent vs component ობიექტებს შორისაც? (სხვა კონტექსტებში ხშირად ერევათ ეს ორი ტერმინი).
